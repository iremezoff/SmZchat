Список консольных команд для управления чатом:
##############################################
############ Для администратора ##############
##############################################

 - Кол-во максимально допустимых смайлов в сообщении
 /setcountsm <КОЛ-ВО>

 - Кол-во секунд, после которых юзер кикается
 /setkick <КОЛ-ВО>

 - Имя бота в текущей комнате
 /setbotname <ИМЯ>

 - Название текущей комнаты
 /setroomname <НАЗВАНИЕ>

 - Назначить баллы пользователю
 /setballs -n <НИК> -b <БАЛЛЫ>

 - Кол-во хранимых в БД сообщений (для каждой комнаты)
 /setcountmsgs <КОЛ-ВО>

 - Очитска логов за заданный период
 /clearlogs -s <ДАТА, ГГГГ-ММ-ДД> -f <ДАТА, ГГГГ-ММ-ДД>
 -s ::: начальная дата очищения логов
 -f ::: конечная дата очищения логов
 Если не указан второй параметр, то он берётся как текущая дата

 - Очистка жалоб за заданный период
 /clearcompls -s <ДАТА, ГГГГ-ММ-ДД> -f <ДАТА, ГГГГ-ММ-ДД>
 -s ::: начальная дата очищения жалоб
 -f ::: конечная дата очищения жалоб
 Если не указан второй параметр, то он берётся как текущая дата

 - Максимальная ширина загружаемой пользовательской фотографии
 /setwidthmax <ШИРИНА, пикселы>

 - Максимальная высота загружаемой пользовательской фотографии
 /setheightmax <ВЫСОТА, пикселы>

 - Максимальный размер загружаемой пользовательской фотографии
 /setsizemax <РАМЗЕР, Кб>

 - Максимальное кол-во хранимых сообщений в личке
 /setmailmsgs <КОЛ-ВО>

 - Кол-во дней, за которые хранятся логи банов
 /setdayslogs <КОЛ-ВО>

 - Кол-во дней, за которые хранятся жалобы на модеров
 /setdayscompl <КОЛ-ВО>

##############################################
############## Для модератора ################
##############################################
 - Бан пользователя:
 /ban -n <НИК> -t <ВРЕМЯ, мин> -r <ПРИЧИНА> -p <ТИП БАНА> -i <IP>
 Все параметры, кроме IP, обязательны

 - Предупреждение пользователей:
 /warn -n <НИК> -m <СООБЩЕНИЕ>

##############################################
######### Для простых пользователей ##########
##############################################

 - Очистка сообщений в чате
 /clearmsgs

 - Обновление сообщений в чате
 /refmsgs

 - Игнорирование пользователя
 /ign <ПОЛЬЗОВАТЕЛЬ>

 - Разигнорирование пользователя
 /unign <ПОЛЬЗОВАТЕЛЬ>

 - Список игнорируемых пользователей
 /myigns

 - Список игнорирующих пользователей
 /meigns

 - Смена комнаты
 /setroom <НАЗВАНИЕ КОМНАТЫ>

 - Выход
 /logout

 - Установка статуса
 /setstatus <СТАТУС>
 Возможные статусы: free, dnd, na, away

 - Установка стиля
 /setskin <СТИЛЬ>
 Возможные стили: установленные в чате

 - Установка типа времени
 /settime <ТИП>
 Возможные типы: hm, hms, ms

 - Установка размещения формы отправки
 /setsend <РАЗМЕЩЕНИЕ>
 Возможные размещения: up, down

 - Установка места появления новых сообщений
 /setmsgs <РАЗМЕЩЕНИЕ>
 Возможные размещения: up, down

 - Установка смайлов
 /setsmile <ПАРАМЕТР>
 Возможные параметры: on, off

 - Установка личного e-mail
 /setmail <E-MAIL>

 - Установка информации о себе
 /setabout <ИНФОРМАЦИЯ>

 - Установка скрытия e-mail
 /hidemail <ПАРАМЕТР>
 Возможные параметры: on, off

 - Изменение пароля
 /setpass -l <СТАРЫЙ ПАРОЛЬ> -n <НОВЫЙ ПАРОЛЬ> -c <ПОВТОРИТЕ ПАРОЛЬ>

 - Добавление жалобы на модера
 /addcompl -n <НИК> -m <СООБЩЕНИЕ>


Команды для всех пользователей доступны также для админов и модеров.
Команды для модераторов доступны также для админов.
