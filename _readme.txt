	Возможности чата:
	_________________________________________________________________
	
	->	Регистрация и авторизация пользователей чата
	->	Защита от регистрации роботами, а также управление этой опцией
	->	Профили пользователей
	->	Мультиязыковая поддержка (только интерфейс основного чата без админки, регистрации и восстановления пароля)
	->	Балловая система пользователей:
			-менее 100 баллов : обычный пользователь
			-100-499 баллов : пользователь с разрешёнными стилями(b,i,u,s,sup,sub,tt,big,cmall)
			-500-699 баллов : модератор
			-700-998 баллов : модератор с возможностью редактирвания правил и смайликов
			-999 баллов : администратор (полный комплекс управления чатом)
	->	Смайлы (ограничитель: разумный выбор)
	->	Возможность отключать смайлы
	->	Возможность посылки приватных сообщений
	->	Обращение к пользователю в чате непосредственно
	->	Статусы пользователя: 
			-Свободен для чата
			-Временно отошёл
			-Недоступен
			-Не беспокоить
	->	Мультиязыковая поддержка (любой может сделать свой перевод любого языка), пока только основного чата
	->	Консоль (администраторские, модераторские и пользовательские настройки, 
		изменяемые с помощью команд, вводимых в строку ввода сообщения)
	->	Возможность выбора цвета текста из более чем 20 цветов
	->	Возможность менять расположение формы отправки (сверху, снизу)
	->	Возможность менять место появления новых сообщений (сверху, снизу)
	->	Возможность изменять тип времени сообщений (час:мин, час:мин:сек, мин:сек)
	->	Автоматический выброс пользователей, не отвечающих на запросы сервера
		в течение определённого промежутка времени
	->	Возможность банить по нику, ip (в будущем по кукам и софту)
		Типы:
			-оставить в чате : возможность оставаться в чате без способности
			 отправки сообщений
			-запретить доступ : пользователь будет каждый раз "кикаться" при
			 попытке входа в чат
			-забить окнами : у пользователя не будут обновляться фреймы
	->	Личка (внутренняя почта)
	->	Возможность игнорировать сообщения других пользователей
	->	Просмотр игнорируемых и игнорирующих пользователей
	->	Комнаты по интересам (комнаты задаёт администратор)
	->	Особенность: невозможность бана модератора другим модератором, чьи баллы меньше
	->	Все действия по бану и разбану, а также истечению банов, записываются в логи,
		которые может просмотреть любой модератор и администратор
	->	Блокировка сообщений, в которых кол-во смайликов превышает их лимит
		(для уменьшения траффика)
	->	Возможность модераторам предупреждать пользователей
	->	Загружать пользовательские фотографии
        ->	Выбор скина
	->	Создание дампа БД чата администратором чата
	->	Отсутствует гостевой вход в чат из-за его беполезности (по моему мнению)

	Разрешённые коды:

		Введите в строку сообщения данный код
		(в скобках указано минимальное кол-во баллов):
		/b 	- жирный	(100)
		/i	- курсив	(100)
		/u	- зачёркнутый	(100)
		/s	- перечёркнутый	(100)
		/sup	- надстрочный	(100)
		/sub	- подстрочный	(100)
		/tt	- телетайп	(100)
		/big	- увеличенный	(100)
		/small	- уменьшенный	(100)
		/beg	- бегущая строка	(500)
		/ips	- все IP пользователей в чате	(500)

	Установка чата:
	_________________________________________________________________
	
	Способ первый:
	1) 	Скопируйте все файлы в определённую директорию.
	2) 	Запустите программу установки скрипта по адресу http://адрес_вашего_чата/install/.
	3)	Измените права доступа к файлу inc/db.php (777), если требуется перезапись параметров.
	4)	Следуйте указаниям программы установки.
	5)	Установите права доступа к директории photos для возможности загрузки пользовательских фото
	6) 	ВНИМАНИЕ: После завершения работы программы установки не забудте УДАЛИТЬ папку install!
		В противном случае, чат может быть легко взломан злоумышленниками!

	Способ второй:
	1)	В директории install имеются 2 файла запросов к БД mySQL: sql.sql и sql_update.sql
		В первом находятся запросы для полной установки или переустановки чата, поэтому если вы решили
		переустановить чат, но оставить текущие настройки или данные, сперва сохраните их с помощью
		дампа БД, иначе они будут потеряны.
		Во втором файле находятся запросы для обновления БД чата для совместимости с предыдущей версией, 
		если таковы имеются. Если версия изменилась, но файл запросов обновления пуст, значит никаких
		изменений в БД не вносилось.
	2)	Скопируйте содержимое одного из файлов (по вашему усмотрению) и вручную произведите установку.

	Общее) Скопируйте в директорию чата fonts файл arial.ttf (или любой другой(ие) файл(ы) TrueType).
        Для пользователей Windows он найдётся в директории C:\Windows\fonts или C:\Winnt\fonts (если ОС установлена на диске C:).
	Для пользователй *nix: спросите у пользователей Windows :)

	ВАЖНО!!! При автоматической установке вы вводите параметры чата. При ручной установке вам дополнительно 
	придётся их вводить либо непосредственно в БД (например, при помощи phpMyAdmin), либо в админке чата.
	Также вам вручную придётся создавать пользователя с правами администратора. Этого нельзя сделать 
	при обычной регистрации, поэтому проделывайте это вручную.

	Настройка чата:
	_________________________________________________________________
	
		Настройка чата производится из админ.панели, доступной 
		администратору (пользователь с кол-вом баллов не менее 999).
		Помимо того, редактировать правила и смайлики
		имеют право пользователи с кол-вом баллов не менее 700.

	_________________________________________________________________
	
	Добавление смайлов
	_________________________________________________________________
	
	1)	Скопируйте нужные Вам файлы смайлков в директорию smiles,
		находящуюся в директории чата
	2)	Зайдите в админ.панель на вкладку "Редактировать смайлики"
		и нажмите на ссылку "Добавить смайлик"
	3)	Введите код смайлика и выберите изображение из списка
		скопированных Вами файлов
